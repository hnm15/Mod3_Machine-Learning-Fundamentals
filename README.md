# Machine Learning Fundamentals

## Overview

This repository contains coursework and projects completed for a Machine Learning Fundamentals course. The repository includes homework assignments covering core machine learning concepts and a group project applying an end-to-end machine learning workflow to a real-world housing dataset.

The work focuses on developing practical skills in:

* Data exploration and visualization
* Data cleaning and preprocessing
* Feature engineering
* Regression modeling
* Model evaluation and selection
* Communicating machine learning results

## Homework Assignments

The `Homework_Assignments` folder contains individual assignments completed throughout the course.

Topics covered include:

* Linear regression
* Polynomial regression
* Model selection using training and testing sets
* Feature engineering
* Transforming datasets with nonlinear features
* Evaluating machine learning models using performance metrics

## Group Project

The `Group_Project_Zillow_Housing_Dataset` folder contains the team's machine learning project using the Zillow housing dataset.

The project follows the machine learning workflow:

1. Problem definition
2. Data exploration
3. Data cleaning and preprocessing
4. Feature relationship analysis
5. Feature engineering
6. Model development and evaluation
7. Final model selection and presentation

For more details, see the README inside the Zillow project folder.
